# 页面优先级审计

目标：任何页面都先把 P0 放在首屏，P3 默认收起或移除。

## 首页 `/`
- P0: 填写出生信息
- P1: 最近记录、最近内容
- P2: 少量后续路径
- P3: 分流规则、产品原则、内容体系、内容证据
- 处理: P0 置顶，P3 收起或删除

## 分析页 `/analyze`
- P0: 出生信息录入
- P1: 提交后结果、已有进度
- P2: 常见辅助入口
- P3: 图片说明、方法论、后续入口说明
- 处理: P0 直显，P3 收起

## 工具中心 `/tools`
- P0: 选择问题线
- P1: 推荐工具
- P2: 其他进入方式、工具包
- P3: 工具矩阵说明、证据层、工具中心角色说明
- 处理: P0 置顶，P3 折叠

## 工具详情 `/tools/[slug]`
- P0: 工具标题、运行器、开始执行
- P1: 必要的运行前确认
- P2: 结果后续承接
- P3: 问法说明、案例、FAQ、推荐、升级、工具包
- 处理: 运行器前移，P3 折叠

## 工具结果 `/tool-result/[sessionId]`
- P0: 结果摘要、继续深问、返回报告
- P1: 免费结果摘要、下一步承接
- P2: 继续看关联工具
- P3: 深度解释、质检、历史上下文、升级、工具包、推荐
- 处理: P0 直显，P3 折叠

## 聊天页 `/chat`
- P0: 追问工作台
- P1: 当前绑定上下文
- P2: 切换专项、更新状态
- P3: 追问规则说明
- 处理: P0 直显，P3 折叠

## 知识库 `/knowledge`
- P0: 专题地图、文章列表
- P1: 快速分析转化
- P2: 更多入口
- P3: 世界易系统层、数据面板、图片说明、订阅
- 处理: 阅读列表前置，P3 折叠

## 案例库 `/cases`
- P0: 案例列表
- P1: 快速分析转化
- P2: 更多入口
- P3: 案例观、案例分类、数据面板、案例纪律、证据层
- 处理: 列表前置，P3 折叠
