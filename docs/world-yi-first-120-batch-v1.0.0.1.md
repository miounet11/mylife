# 世界易首批 120 篇执行批次 v1.0.0.1

这份文档不是远景，而是执行图。

它解决的是一个更现实的问题：

`十卷主书已经初版完成，接下来前 120 篇到底怎么批量生产，才不会重新写散。`

---

## 1. 批次原则

首批 120 篇不是平均摊开，而是按“先立母语，再立六域，再立应用，再立全球，再立案例，再立治理”的顺序推进。

这样做的原因很简单：

- 没有母语，后面都会散
- 没有六域，用户进来找不到现实问题入口
- 没有应用，体系会显得只会讲宏大概念
- 没有全球华人层，世界易的世界性不成立
- 没有案例，判断秩序没有证据层
- 没有治理，内容量一上来就会失控

---

## 3. 当前判断

如果只看“有没有内容”，世界易已经不缺方向。

现在真正缺的是：

- 按批次推进
- 按卷次回挂
- 按页面入口分发
- 按质量门槛筛选

所以这 120 篇不是简单写满，而是要写成：

- 能反向支撑十卷主书
- 能直接挂进站内页面
- 能形成全球华人和英文入口
- 能持续升级到 300、800、2000 篇

---

## 4. 执行纪律

每篇进入首批 120 的内容，至少必须满足：

1. 能挂回十卷中的某一卷
2. 有主判断
3. 有判断依据
4. 有动作建议
5. 有风险提醒
6. 有页面入口去承接它

不满足这六条，就不算真正进入世界易首批执行层。
