## 1. 定位

### 1.1 一句话定位

`用现代视觉语言，把传统命理和易学知识解释成可理解、可分享、可进入测算的结构图。`

### 1.2 与世界易的关系

传统术数图库不是要把人生K线退回“玄学百科”，而是把用户最熟悉的传统入口重新翻译为世界易的判断秩序：

- 先普及概念
- 再说明结构
- 再解释边界
- 再导向个人测算
- 最后进入报告、工具、内容与复访

所有图片最终都要回到世界易总式：

`结构 -> 时位 -> 环境 -> 动作 -> 风险 -> 复盘`

### 1.3 三类用途

| 类型 | 作用 | 主要位置 |
| --- | --- | --- |
| 教材图 | 解释基础概念、关系、流程、图谱 | 知识页、世界易主书、工具说明 |
| 测算转化图 | 把传统入口导向个人分析 | `/analyze`、报告页、工具页 |
| 传播图 | 让用户收藏、转发、分享 | 小红书、抖音、快手、视频号、朋友圈、繁体中文社群 |

## 2. 安全边界

传统术数内容容易滑向恐吓、决定论和过度断言。1000 张图必须遵守以下边界。

### 2.1 统一表达边界

- 不说“必然发财、必然离婚、必然有灾”。
- 不说“某五行就是某种命运”。
- 不说“某手纹、某面相就一定代表某人格或人生结果”。
- 不做医疗、法律、投资、婚姻结果的确定性判断。
- 不用“吓你改命”的销售方式。
- 不把传统概念写成神秘权威，而要写成结构解释和文化普及。

### 2.2 面相、手相、摸骨边界

面相、手相、摸骨只能做文化普及和历史知识图：

- 使用抽象线稿、古籍风格示意、非真实人物、非真实用户照片。
- 不对真实人脸、真实手掌、真实骨骼做自动判断。
- 不从外貌推断人格、健康、命运、财富、婚姻、职业能力。
- 不使用真实名人照片、用户自拍或敏感身体特征图。
- 必须在文案中强调“传统文化知识，不等同于现实结论”。

### 2.3 风水边界

风水图可以解释空间秩序、动线、采光、恢复位、冲突源，但不能替代：

- 建筑安全
- 消防规范
- 医疗建议
- 法律合规
- 专业室内设计

### 2.4 起名与字五行边界

康熙字典、笔画、字义、部首、音韵、五行判断都只能作为参考：

- 不承诺名字改变命运。
- 不把单字五行当成唯一依据。
- 必须同时考虑结构、语义、声音、场景、长期使用成本。
- 简体、繁体、异体字、康熙笔画需要分开说明。

## 21. 质量检查

每张图上线前必须检查：

- 是否解释一个明确概念，而不是堆砌神秘符号。
- 是否有大标题安全区。
- 是否避免生成难以阅读的中文小字。
- 是否避免恐吓、决定论和“必然结果”。
- 是否能导向测算、工具、知识或案例入口。
- 是否和世界易方法一致：结构、时位、环境、动作、风险、复盘。
- 是否注明文化普及边界，尤其是相学、手相、摸骨、风水、起名。
- 是否有简体标题、繁体标题、alt 文案和路由。
- 是否适合移动端裁切。
- 是否适合长期复用，不会因年份变化失效。

## 22. 结论

1000 张命理与易学图库可以做，而且应该做。

但它不能做成“玄学符号堆砌”。正确方向是：

- 用 1000 张图建立传统知识入口。
- 用世界易把传统概念翻译成现代判断语言。
- 用测算、报告、工具和内容矩阵承接用户兴趣。
- 用清晰边界避免恐吓、迷信化和决定论。

这套图库完成后，人生K线可以同时拥有：

- 产品说明图
- 世界易结构图
- 命理教材图
- 传统术数普及图
- 社媒传播图
- 工具转化图

最终用户看到的不应该是“神秘感”，而应该是：

`原来这些传统概念可以被讲清楚，而且可以进入我的现实判断。`
