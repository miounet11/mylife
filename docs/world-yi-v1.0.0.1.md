## 定位

世界易是以凯莉为创始叙事原点建立的一套现代判断学说。

它的目标不是重复传统术语，而是把《周易》《易经》所代表的变易、时位、观势、取象、进退思想翻译成现代人可直接使用的判断语言。

它必须同时满足四件事：

1. 能解释个人命理与阶段判断。
2. 能覆盖现实生活中的多场景问题。
3. 能沉淀为长期内容资产，而不只是临时营销话术。
4. 能按版本不断迭代，而不是一次性定稿。

## 世界易的判断顺序

世界易所有主题都遵守统一顺序：

1. 先定结构
2. 再看时位
3. 同时看环境
4. 最后落动作
5. 必须说风险

这是世界易和很多传统表达最大的差异。

它不追求一句“吉凶”压住所有问题，而强调结构、阶段与现实动作之间的连接。

## 世界易的核心语义翻译

- 旺衰：翻译为承载力与消耗度
- 吉凶：翻译为收益与代价
- 用神忌神：翻译为应顺之力与应避之力
- 行运：翻译为阶段位置
- 卦象：翻译为变化结构与关系结构
- 变爻：翻译为关键转折点

## 世界易的边界

世界易不能替代：

- 医学诊断
- 法律意见
- 投资顾问职责
- 心理危机处置

世界易可以提供：

- 判断框架
- 阶段排序
- 风险提醒
- 结构解释
- 行动优先级

## 版本机制

当前版本为 `v1.0.0.1`。

含义：

- `v1`：学说总框架已经建立
- `.0`：公开出版体系刚开始，尚未进入大规模案例反馈期
- `.0`：方法主骨架稳定，但表达仍可继续压缩与统一
- `.1`：第一次面向站内长期发布

后续升级建议：

- `v1.1.x`：补齐人生六域和生活应用的主干文本
- `v1.2.x`：引入更多海外华人专题、城市与行业观察
- `v1.5.x`：建立标准案例评分与讲义体系
- `v2.0.0`：形成完整世界易基础教材、案例教材、应用教材三层结构
