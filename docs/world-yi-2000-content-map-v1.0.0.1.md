## 发布顺序建议

### Phase 1

- 先做 `20` 篇世界易基础文本
- 建立总论、方法、六域和应用入口

### Phase 2

- 扩到 `100` 篇
- 形成知识页、案例页、城市页、行业页互链

### Phase 3

- 扩到 `300` 篇
- 建立世界易基础教材、案例教材

### Phase 4

- 扩到 `800` 篇
- 形成稳定搜索入口与专题路径

### Phase 5

- 扩到 `2000` 篇
- 完整覆盖世界易学说、案例、应用与全球传播场景

## 编辑纪律

所有世界易内容必须满足：

1. 有主判断
2. 有判断依据
3. 有可执行动作
4. 有风险提醒
5. 有版本归属

不满足这五条的内容，不进入公开发布层。
